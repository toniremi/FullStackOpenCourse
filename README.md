# FullStack Open Course (University of Helsinki)

This repository will contain my solutions to the exercises of the [Full Stack Open course provided by the University of Helsinki](https://fullstackopen.com/en/about).

> This course serves as an introduction to modern web application development with JavaScript. The main focus is on building single page applications with ReactJS that use REST APIs built with Node.js. The course also contains a section on GraphQL, a modern alternative to REST APIs.

## Repository Structure

The repository is organized by parts, represented in directories. Each directory corresponding to a specific part of the course (e.g., `part0`, `part1`).

Each part is further divided into specific folders which will contain part specific exercises and projects.

```Text
./FullStackOpenCourse
├── part0/
├── part1/
│   ├── courseinfo
│   ├── unicafe
│   └── anecdotes
├── part2/
│   ├── courseinfo
│   ├── phonebook
│   └── countries
└── ...
```
